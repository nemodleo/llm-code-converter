    🔄 Three-Stage Pipeline

    01

    Dependency Analysis

    Call Graph 기반 정적 분석을 통해 함수 간 의존 관계를 파악하고, Proworks 5에 적합한 스켈레톤 코드와 데이터 클래스를 자동 생성

    Tree-sitter javalang Static Analysis
    📊 95% Coverage
    02

    LLM Code Conversion

    LLM과 RAG 기법을 활용하여 레거시 함수를 Proworks 5 API에 매핑하고, 외부 문서를 실시간으로 참조하여 변환 정확도 향상

    LLM RAG FAISS
    🎯 80% Accuracy
    03

    Self-Refinement

    Reflexion 방법론을 활용하여 LLM이 자기 출력을 재검토하고 개선하는 Agentic 방식으로 코드 품질을 지속적으로 향상

    Reflexion Self-Feedback Iterative Improvement
    🔄 3x Iterations
