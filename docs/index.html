🔄 Three-Stage Pipeline

01

Dependency Analysis

호출 그래프(Call Graph) 기반 정적 분석을 통해 함수 간 의존 관계를 파악하고, 변환에 필요한 VO 클래스와 스켈레톤 코드를 100% 정확도로 자동 생성합니다.

Tree-sitter Static Analysis
02

LLM Code Conversion (RAG)

LLM과 RAG 기법을 활용하여 레거시 함수를 Proworks 5 API에 매핑합니다. 외부 문서를 실시간으로 참조하여 API 변환 정확도를 98%까지 끌어올립니다.

LLM RAG FAISS
03

Self-Refinement (Reflexion)

Reflexion 방법론을 기반으로 LLM이 자기 출력을 재검토하고 개선하는 Agentic 방식으로, 코드의 논리적 오류와 규칙 위반을 스스로 수정하여 품질을 높입니다.

Reflexion Self-Feedback
